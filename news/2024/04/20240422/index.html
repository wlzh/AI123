🦉 AI新闻

🚀 全球AI投资减少，生成式AI面临挑战

摘要：2023年，随着投资者对于人工智能（AI）领域的热度有所降温，全球对AI的投资呈现持续下降趋势。斯坦福大学以人为本人工智能研究所的报告指出，无论是私人投资还是企业投资，AI行业的资金流入均比去年有所下滑。虽然生成式AI初创企业的投资额有显著增长，但整体融资环境趋紧。投资者开始对AI初创公司的高估值和长期盈利能力抱有怀疑，特别是对于运营成本高但收入低的公司，如Stability AI。同时，B端企业对生成式AI技术带来的生产力提升也持怀疑态度。

🚀 Command R + 成首个击败GPT-4的开源模型

摘要：近日，开源模型Command R + 创历史首次击败了GPT-4，登陆HuggingChat可免费尝试。该模型以2.3万的投票量，在LLM竞技场中与GPT-4的早期版本并列第7位。Cohere的机器学习总监Nils Reimers表示，Command R + 的真实优势在于RAG和工具使用能力，这在竞技场未得以体现。它由Transformer架构共同作者之一Aidan Gomez领衔，拥有1040亿参数支持10种语言，专为企业高效精准的用例构建，并优化了多语言支持和工具使用功能，有效降低成本。

🚀 微软WizardLM-2超越GPT-4后突然下架

摘要：微软最近公布并开源了一款新的大型语言模型WizardLM-2，该模型性能超过了GPT-4和Mistral，引起了广泛关注。不过，在该模型上架后不久，微软突然无预警地删除了所有项目和代码文件，原因是开发团队“不太熟悉新的模型发布流程，忘记进行幻觉测试”。目前，开发团队正在紧急进行测试，计划测试完成后尽快重新发布。据官方简介，WizardLM-2的性能远超目前市场上的其他模型，这使其成为了业内关注的焦点。尽管目前模型已下架，但该事件仍引发了业界对微软在人工智能领域的技术进步和管理流程的关注。

🚀 微软Word新增集成必应搜索的Copilot功能

摘要：微软最近更新了Microsoft 365的路线图，宣布计划于4月底在Microsoft Word中进一步整合Copilot功能，通过内置的必应搜索引擎直接回答用户提出的问题，而无需离开Word应用。新增的“Microsoft Copilot (Microsoft 365)：集成必应搜索”的功能允许用户直接在Word聊天中提问，Copilot利用必应搜索提供答案。这项服务向Copilot Pro和Copilot for MS365用户开放，后者为一款人工智能助理工具的订阅计划，月费30美元（约217元人民币）。

🚀 谷歌重组AI部门整合DeepMind和Google Research

摘要：谷歌最近宣布重新整合其AI部门，将DeepMind和Google Research等团队合并为一个新部门，旨在加强AI的研发和商用推广。此次整合会让DeepMind主要专注于AI模型的构建，而Google Research则将其重点转移到基础计算机科学研究上。此外，谷歌也在整合其Responsible AI团队，目的是强化模型开发、产品创新与用户体验之间的互动，并对AI功能的发布实施更为严格的要求，同时增加对AI的投资测试和评估。

🚀 GlitchHunter：显著提升大型语言模型准确性

摘要：近日，由华中科技大学和南洋理工大学联合的研究团队提出了GlitchHunter研究，首次对大型语言模型（LLM）中的故障词元进行了全面分析。GlitchHunter通过收集并分类大量故障词元，使用迭代聚类技术识别并处理这些词元，显著改善了LLM的输出质量。测试结果显示，GlitchHunter在真阳性率、精确度和召回率上均优于现有方法，同时减少了处理时间和词元数量，展现了其在提高模型可靠性和输出质量方面的潜力和实用性。

🚀 清华发布最全面LLM综合能力评测报告

摘要：2023年的「百模大战」中，清华大学基础模型研究中心与中关村实验室联合研制了SuperBench评测框架，对国内外14个具有代表性的大模型进行了全面综合能力评测。其中，GPT-4和Claude 3表现出色，位居国际前列，而国内的GLM-4和文心一言4.0也表现亮眼，紧随其后，显示出国内外大模型技术差距正在逐渐缩小。此次评测覆盖了语义、代码、对齐、智能体和安全五个评测大类，目的在于推动大模型技术、应用与生态的健康发展。

🚀 TikTok开发AI语音库功能，简化语音生成

摘要：据IT之家4月20日报道，用户@AssembleDebug发现TikTok正在开发一项新技术，允许用户通过简短录音快速创建属于自己的声音加入到“TikTok语音库”。这项功能通过解包最新版TikTok应用得以发现，目前尚处于开发阶段，发布时间未定。用户需录制一段文本读音即可生成AI版本声音，在TikTok视频中使用文本转语音功能。TikTok采取了措施保护用户隐私，允许随时删除AI语音以防盗用。
