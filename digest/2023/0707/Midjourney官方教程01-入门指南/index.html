前言

Mid journey是一款2022年3月面世的AI绘画工具，创始人是David Holz。

只需要输入文字，MJ就能给出对应的图片。2022年8月，游戏设计师杰森·艾伦（Jason Allen）在没有绘画基础的情况下，凭借Mid journey加PS的绘画作品《太空歌剧院》，获得美国科罗拉多州新兴数字艺术家竞赛一等奖，让Mid journey一战成名；

Mid journey官方中文版已经开启内测，搭载在QQ频道上，每周一、周五下午6点统一开放，达到一定人数后会关闭入口；加入频道后，在常规创作频道中输入 /想象 + 生成指令，即可召唤 Mid journey 机器人进行作画。

鉴于使用MJ的人越来越多，迫切需要一份权威、全面、实时的中文教程；

Mid journey官方文档中文翻译版

官方文档的地址：https://docs.midjourney.com/docs/quick-start

1：入门指南

1.1：快速开始

学习如何在Discord上使用Mid journey机器人，用简单的文本提示，创建自己的图像。

行为准则 ：

不要做恶 ；

不要用我们的工具制作可能引起不安或引起争议的图像，包括血腥和成人内容；

对其他人和团队保持尊重。

1.2：加入Discord

前往Midjourney.com，选择“加入Beta”，或直接进入MidjourneyDiscord。

Discord账户 在加入MidjourneyDiscord服务器之前，您必须拥有已验证的Discord登录。

1.3：寻找新手频道

在Mid journey官方服务器上

选择左侧边栏中可见的任何新手（newbies-#）频道。

在其他服务器上

您可以在已邀请Mid journey机器人的任何服务器上生成图像。查找有关在您的服务器上使用机器人的说明。

1.4：使用/imagine命令

关于Discord命令

使用命令与Mid journey Bot在Discord上进行交互。命令用于创建图像，更改默认设置，监视用户信息并执行其他有用的任务。

/imagine命令：用简短的文本描述（称为提示）生成唯一的图像。

使用/imagine

  1. 输入/imagine提示：或从斜杠命令弹出窗口中选择/imagine命令。

  2. 在“提示”字段中输入您想要创建的图像的描述。

  3. 点击回车键发送您的消息。

Mid journey机器人将生成一个弹出窗口，要求您接受服务条款。在生成任何图像之前，您必须同意服务条款。

说明：

如果在输入/imagine命令时没有弹出窗口，请尝试退出登录，更新Discord应用程序并重新登录。

命令只能在机器人频道中使用。命令无法在常规频道（如#trial-support）中使用。

1.5：处理工作

Mid journey机器人需要大约一分钟的时间，来生成四张图片。

生成图像将激活免费的Mid journey试用。试用用户可以在需要订阅之前，大约发出25个指令（遗憾的是，现在已经不免费了）

指令是使用Mid journey机器人执行的任何操作。指令包括使用/imagine命令创建图像网格、放大图像或创建图像变体，所有这些都会使用您的免费试用额度。

使用/info命令检查您的快速剩余时间，以查看您剩余的试用时间。

1.6：提升像素（Upscale）或创建变体（Variations)

在初始图像网格生成完成后，会出现两行按钮：

U1 U2 U3 U4

U 按钮：可以将图像放大，生成所选图像的更高像素的版本，并添加更多细节。

V1 V2 V3 V4

V按钮：根据所选网格图像，再创建4轻微变化的变体。点击其中一个变体，会生成一组新的4张图片，新图与选择的图像的整体风格类似

1.7 ：评价 图像

在使用放大 图像后，将 出现一组新的选项：

制作变体（Make variations） 轻微升级重做（Light Upscale Redo） Beta 升级重做（Beta Upscale Redo） ↗ 网 页 （Web ）

制作变体 ：创建一个升级图像的变体，并生成四个选项的新图像组 。

Beta/Light Upscale Redo：使用不同的升级模型重新进行升级。

Web ：在Midjourney.com上打开您相册中的图像 。

点击Midjourney网站或Discord上任何一张经过提高分辨率处理后的图片下面的笑脸按钮来评价它们 。

每天，前1000名 图片评分者将获得一小时免费快速模式额度 。

根据您个人风格和观点对自己或他人的工作进行评分 。喜欢外观、努力、颜色、概念或主题吗？请给它一个 ！

1.8：保存您的图片

单击 图像，以全尺寸打开，然后右键单击并选择“保存 图像”。

在移动设备上，长按 图像，然后点击右上角的下载 图标 。

所有 图片都可 以立即在midjourney.com/app上查看 。登录Discord以查看 。

1.9：订 阅计划

试用用户大约可 以免费使用25个指令 。目前，指令不会过期，但也不会 自动续订 。请在任何Bot频道 中使 用/subscribe命令 以制作更多图像 。

有关价格和更多信息，请转至订阅计划 。

双重检查 请确保您正在使用现有的Discord帐户登录 。因为很容易意外地创建一个新帐户 。

*社区准则 *

Mid journey是一个默认开放的社区。为了让平台对尽可能多的用户都可用和友好，内容必须符合 PG- 13 标准 。

请友善相待并尊重彼此和工作人员。不要创建具有固有不敬、攻击性或其他虐待性质的图像或使用文本提 示 。任何形式的暴力或骚扰都将被禁止 。

禁止发布成人 内容或血腥场面，请避免制作视觉上令人震惊或令人不安的内容 。我们会 自动屏蔽某些文本输入 。

未经他人许可，不得公开转载他人创作物 。

在分享时要小心谨慎 。可以在 Mid journey 社区之外分享您的创意，但请考虑其他人如何看待您的内容 。

违反这些规定可能导致被禁止使用我们的服务 。我们并非不讲道理，每个人的行为应该恭敬礼貌 ，否则将失去使用服务权利 。

这些规则适用于所有 内容，包括在私有服务器 中制作、使用私密模式 以及与 Mid journey 机器人进行直接消息交流所产生的图像等等 。请参阅 Discord #rules 频道获取最新信息。

注释

什么被认为是血腥的？血腥包括人类或动物分离的身体部位 图像、食人族行为、暴力 （例如射击或轰炸某人的图像 ）、畸形身体、断肢、瘟疫等 。

什么是不适宜工作场所或成人内容 ？避免裸露，性器官，对裸露胸部着迷，淋浴中或在厕所上的人，性臆想，恋物癖等 。

其他冒犯性内容、其他事情可能被视为种族主义、同性恋歧视、令人不安或以某种方式贬低社区而被认为具有冒犯性或虐待性。这包括裸体儿童和名人或公众人物的冒犯 图片。内容指南将不断审查，并随着Mid journey社区的发展进行修改。

自动禁止关键词

在促进艺术表达和创建一个安全、受欢迎 的社区之间找到平衡是具有挑战性的。每个人都有权利对什么图像美丽、引人入胜或启发性发表意见，而对于哪些 内容是 冒犯、不适当或有害的，也存在许多个体解释 。我们也认识到，艺术长期以来一直推动着现状，挑战社会惯例，激励讨论并引起变革 。

在制定Midjourney社区准则时，我们考虑 了许多因素 。作为一种新兴技术，在考虑更广泛公众如何看待和介绍这项 技术方面非常重要 。

Midjourney比画笔或铅笔更具参与性，并不希望参与创造色情、冒犯或有害的图像 。然而最重要的是我们希望为所有用户创建一个友好和包容的社区。为此，我们已经建立了禁止平台上某些行为和语言的准则之一就是禁止使用那些 曾被用于针对我们社区成员造成伤害或冒犯的单词和短语 。

我们理解到所做 出的平衡和禁止某些单词的决定可能不会得到所有人的欢迎 。但我们希望您尊重我们为所有人创建积极、支持性社区所做出的努力。我们鼓励您以创造性的方式表达自己，但请在考虑他人时保持尊重和体贴。

为什么这个词被禁用 ？

一些单词和短语之所 以被禁用，是因为它们产生的内容被广泛认为是不适当、冒犯或有害的。一些被禁用的单词看起来很安全，但却经常产生违反内容指南的图像 。

请记住 ，Midjourney机器人不能像人类那样理解语言。

例如，“桃子”表情符号，目前是一个被禁止使用的单词，提示“美丽 果园”将触发该过滤器 。然而，没有人认为桃树果园不雅或在工作中无法查看 。然而，“ ”表情符号也是常见的屁股俚语，并经常与NSFW图片相关联，因此提示“美丽 果园”的实际结果 比你最初想象的更具 问题性 。

多次违反服务条款的用户可能会受到社区管理员警告、暂停访 问机器人或者被封锁服务等处罚。

自我监管和报告

偶尔会有提示无意中产生非PG- 13内容的情况 。请使用表情符号或右键单击选择应用程序 ，然后点击取消作业来自我监管这些图像 。

用户可以通过右键单击选择应用程序 ，然后在Discord或网站上选择“举报作业”来标记任何图像，以供审核员审查；

也可以在图片下方选取…并点击“举报”。
