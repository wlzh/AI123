20231220 OpenAI推出安全准备框架，解决AI模型的灾难性风险问题


  • By AI123
  • December 20, 2023 - 2 min read



🦉 AI新闻

🚀 OpenAI推出安全准备框架，解决AI模型的灾难性风险问题

摘要：OpenAI正式宣布成立一个名为"Preparedness"的团队，旨在系统化其安全思维。他们制定了一个框架来解决最先进模型的安全性问题，并成立了一个咨询小组，负责审查安全报告。OpenAI希望通过这一举措，能够在部署新技术之前识别和量化潜在风险，并制定相应的应对措施。他们还将与其他安全政策团队密切合作，以确保模型的安全性。这一举措被视为OpenAI在引导AI向符合人类价值观和安全标准的方向发展所做的努力。

🚀 百川智能开放搜索增强的Baichuan2 Turbo系列API，大规模文本检索更高效

摘要：百川智能在Baichuan2 Turbo系列API发布会上宣布，新增搜索增强的功能，提升大模型的文本规模至相当于1亿汉字的5000万tokens。用户可根据提示词在海量文档中检索相关内容，并将文档与提示词一起放入长窗口中，节省推理费用与时间成本。用户可上传文本资料创建专属知识库，官网大模型已支持多种文本上传和URL输入。该功能将大规模文本检索变得更加高效。

🚀 科大讯飞推出AI智能无线机械键盘D1，支持多种智能功能

摘要：科大讯飞推出AI智能无线机械键盘D1，搭载讯飞星火大模型，售价599元。该键盘支持AI智能生产PPT、问答、写作、绘图等功能，还有多项免费功能。键盘采用84键紧凑布局，配备PBT键帽和佳达隆矮红轴，支持蓝牙、2.4G、有线三模连接。此款键盘可在多系统间自动切换。总体评分为85分。

🚀 华为联合泰国数字经济与社会部举办2023年华为云AI峰会，签署数字化转型合作备忘录

摘要：华为云与泰国气象局合作，打造了精度超过传统预测方法的AI大模型，提升了预测速度。华为还发布了泰语大模型，结合专业知识和海量泰文数据，降低大模型使用门槛，加速泰国实现数智化升级。泰国常受热带气旋影响，预测台风未来路径从5小时缩短到10秒，助力农业和旅游业发展。华为与泰国的合作将在AI技术发展、行业应用、生态繁荣、人才培养等方面展开，推动泰国从AI使用者向创造者的转变。

🚀 GitHub Copilot Chat扩展现可为用户提供建议命名

摘要：微软推出付费GitHub Copilot Chat扩展，为Visual Studio Preview用户提供AI建议变量、方法和类名称的功能。该扩展不仅仅是提供一个名称，而是根据用户的代码风格，提供代码库无缝适合的标识符建议。使用这一工具，用户只需选择一个标识符并右键点击，即可看到个性化的命名建议列表。使用此功能需要成为GitHub Copilot的订阅者，费用为个人用户每月10美元或每年100美元，Copilot Business用户每月19美元，Copilot Enterprise用户每月39美元。用户需安装最新的Visual Studio Preview版本，并更新GitHub Copilot Chat扩展后启用重命名建议选项。

🔥 2023年的生成AI进展和2024年的展望

2023年，ChatGPT的发布引起了人们对人工智能和生成式人工智能的广泛关注。这一年被称为"AI夏季"，风险投资公司向生成式人工智能领域投资了超过360亿美元。人们对ChatGPT的能力感到惊讶，并开始探索不同的提示技术。与此同时，研究人员也在不断推出新的生成式人工智能模型，如LLaMA、Jurassic-2、GPT-4、Claude、Falcon-40B、Bard和BloombergGPT。

🔥 GitHub年度回顾：个性化AI生成的代码视频

2023年，GitHub Wrapped项目的团队在GitHub上向开发者们送上了一份特殊的礼物。他们开发了一个名为Year in code的AI生成视频项目，通过GitHub用户的数据生成个性化的回顾视频。项目利用了LangChain、OpenAI GPT-4-turbo、Remotion等工具和技术，实现了用户认证、数据获取、视频生成和播放等功能。用户可以在客户端观看视频，并且可以下载视频进行分享。这个项目充分利用了最新的技术，为开发者们带来了全新的体验。
