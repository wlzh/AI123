🦉 AI新闻

🚀 微软推出Copilot Pro高级订阅服务，扩展适用范围到更多设备和应用

摘要：微软为其Copilot助手推出了新的高级订阅服务Copilot Pro，该服务每月收费20美元，支持Windows PC、Web、App使用，并即将登陆手机。Copilot Pro付费用户还能够在多个设备上体验到支持Copilot的Microsoft Office软件，并获得OpenAI最新模型和更快的速度和性能。此外，Copilot Pro还推出自定义GPT功能，支持针对特定主题的定制，并为小型企业推出Copilot for Microsoft 365服务。

🚀 OpenAI ChatGPT Enterprise用户达15万，预计月收入900万美元

摘要：OpenAI首席运营官透露，ChatGPT Enterprise目前已有15万用户，260家企业积极使用该服务。ChatGPT Enterprise专为大型企业设计，据称"因客户而异"的价格每月为每人60美元，预计将给OpenAI带来900万美元的收入。自2023年以来，OpenAI为了支撑财务状况陆续推出多项付费方案，估计去年创造了16亿美元的营收。OpenAI还推出适用于中小企业的ChatGPT Team和GPT Store服务，希望吸引企业订阅相关版本的ChatGPT服务。

🚀 AI操作系统启动自己的副本，OpenAI开源项目Open Interpreter融合多模态大模型

摘要：通过OpenAI开源项目Open Interpreter，AI已经可以与操作系统结合，实现自动化任务的执行。这一开源项目已迅速积攒3.7万星，并在最新版本中将多模态大模型作为核心组件，实现了自动发送邮件、编曲软件作曲等功能。同时，团队还计划开发AI时代的新型计算机架构——语言模型计算机(LMC)，并成功吸引了200多名开发者加入该项目。这一动态尽管未见GPT-4的具体成果，但仍具有一定影响力和新颖性。

🚀 谷歌测试发现大语言模型纠错能力有限，建议使用小型专用模型监督

摘要：谷歌研究院使用自家BIG-Bench基准测试创建了"Big-Bench Mistake"数据集，评估市面上语言模型的出错概率和自我纠错能力。测试结果发现，目前最先进的大语言模型的自我纠错能力相对有限，只能找出52.9%的逻辑错误。谷歌建议使用小型专用模型监督大型模型，以提高纠错效率和降低AI部署成本。这项研究对于改进语言模型的纠错能力具有重要意义。

🚀 IMF预测人工智能将对全球40%的工作产生重大影响

摘要：国际货币基金组织（IMF）预测人工智能（AI）将对全球近40%的工作产生重大影响，特别是在发达经济体中的工作面临更大的影响。IMF总裁警告称，AI可能加剧收入不平等，具体取决于它对高收入人群的影响。她呼吁各国政策制定者采取措施减缓这一趋势，并建立社会安全网和再培训计划来帮助处于风险中的工人。IMF的预测与科技行业最近的趋势相吻合，科技巨头们在成本控制和AI应用方面仍在进行裁员。
