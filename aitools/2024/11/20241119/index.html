✨ 1: Chatbox

Chatbox是一个开源的桌面客户端，支持多种语言模型，适用于Windows、Mac和Linux系统。

Chatbox 是一款开源的桌面客户端，旨在为用户提供便捷的 AI 助手体验。它支持多种大型语言模型（LLM），如 ChatGPT、Claude 和其他前沿的 AI 技术，能够在 Windows、Mac 和 Linux 平台上运行。无论是日常通讯、工作协作，还是 API 调试，Chatbox 都为用户提供了强大的功能和友好的界面。

地址：https://github.com/Bin-Huang/chatbox

✨ 2: AI Travel Agent

AI旅行代理人利用LangGraph构建智能旅行助手，支持航班查询、酒店预订及个性化邮件发送。

AI旅行助手是一个智能旅行助手项目，基于LangGraph技术，利用多种语言模型（LLMs）来处理旅行相关任务，如查找航班、预订酒店和发送个性化电子邮件。此助手能够与用户互动，调用必要的工具，提供无缝的旅行规划体验。

AI旅行助手致力于简化旅行规划过程，让用户轻松便捷地获得信息及服务，提升整体旅行体验。可以在用户需要航班信息、酒店选择及相关服务时提供实时帮助，是现代数字旅行时代的一个有力助手。

地址：https://github.com/nirbar1985/ai-travel-agent

✨ 3: LLaMA-Mesh

LLaMA-Mesh通过语言模型实现3D网格生成，支持文本与3D数据的统一处理。

LLaMA-Mesh 是一个将大语言模型（LLM）和三维网格生成相结合的创新模型。该模型通过将三维网格的数据（如顶点坐标和面定义）以纯文本的形式表示，使得LLM能够直接处理三维网格的生成与理解。这一方法利用了LLM中嵌入的空间知识，从文本来源（例如三维教程）中获取信息，实现了文本与三维网格生成的统一。

LLaMA-Mesh 提供了一种新的方法，将文本生成与三维模型创建结合在一起，使得三维内容生成变得更为直观和高效。

地址：https://github.com/nv-tlabs/LLaMA-Mesh

✨ 4: NSFW Detector

NSFW Detector是一款基于AI的NSFW内容检测工具，支持多种文件类型，确保数据安全。

NSFW Detector 是一个用于检测不适宜内容（NSFW）的工具，基于 Falconsai/nsfw_image_detection 模型，使用的是 google/vit-base-patch16-224-in21k。该工具的主要特点包括：

内容审核系统：用于社交媒体、论坛等平台的内容审核，过滤不适宜的图片、视频等。 应用程序开发：集成到应用中，帮助开发者在内容上传功能中添加NSFW检测。 教育和培训：为教育机构提供用户生成内容的安全性检测，保护学生免受不当内容的影响。 企业内容管理：在企业内部信息交流中，监控分享的文件是否包含不当内容。 自动化内容监控：用于自动化工具中，监测大量内容的适宜性，提升内容审核的效率。

该工具支持多种文件类型，包括图片、PDF、视频和压缩包中的文件。使用者可以通过Docker快速启动API服务器，或利用内置的Web界面进行检测。同时，VX.link还提供了公共API，便于进行文件检测。

地址：https://github.com/tmplink/nsfw_detector

✨ 5: X-Portrait Nodes

X-Portrait Nodes 可通过输入视频和参考图像，动画化人像，适用于 ComfyUI。

X-Portrait Nodes是一个用于动画化肖像的工具，运行在ComfyUI平台上。用户可以使用一段视频和一张参考图像来生成动态肖像。

X-Portrait Nodes提供了一种创新的方式，将静态图像与动态视效结合，拓展了各种创作和表达的可能性。

地址：https://github.com/akatz-ai/ComfyUI-X-Portrait-Nodes
