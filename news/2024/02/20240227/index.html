🚀 Mistral发布语言大模型Mistral Large

摘要：Mistral Large 是 Mistral AI 公司最新发布的旗舰语言模型，具备顶尖水平的推理能力。它主要被设计用于处理复杂的多语言推理任务，比如文本理解、转换和代码生成等。此外，该模型在多个常用的基准测试上都取得了强劲的成绩，位居全球第二（紧随 GPT-4 之后），非常适合于通过 API 方式广泛提供使用。

体验地址：https://chat.mistral.ai

🚀 中央广播电视总台推出AI动画片《千秋诗颂》

摘要：由中央广播电视总台制作的中国首部文生视频AI动画片《千秋诗颂》于2月26日在CCTV-1开播，该动画片以中华经典诗词为基础，通过AI人工智能技术展现独特中国审美和文化内涵。《千秋诗颂》依赖“央视听媒体大模型”与其他AI技术，转化200多首教材诗词为国风动画片，其中美术风格融合了水墨画与工笔画特质。全系列共26集，每集7分钟，全部采用AI技术辅助制作，标志着在传统文化传承与科技结合领域的一大进步。

🚀 ChatGPT在法律领域计算律师费遭法官质疑

摘要：纽约城市的Cuddy律师事务所使用ChatGPT评估律师费，并要求每小时600美元的律师费，但被联邦地区法官Engelmayer驳回，称这一费用要求过高且缺乏说服力。法官指出，由于ChatGPT未能明确其数据来源和输入内容，无法判断其提供的信息的真实性和相关性。此前也有律师因依赖ChatGPT提供的虚构法律案例而受到质疑。最终，Cuddy律师事务所的律师费要求被裁定减半至53,050美元。法官建议，除非ChatGPT的可靠性有了根本性的改变，否则应避免在费用申请中引用该工具。

🚀 全球首例AI服务侵权案在广州获生效判决

摘要：广州互联网法院对一起生成式AI服务侵犯著作权案件做出生效判决，标志着全球首例此类案例。案件中，被告公司提供的AI服务侵犯了原告奥特曼作品的复制权和改编权。原告获授权持有奥特曼系列形象的著作权，而被告公司通过其AI服务生成的奥特曼形象，与原告拥有的形象实质性相似，最终法院判决被告向原告赔偿10000元。这一判决继之前北京互联网法院关于AI生成图片著作权案后，进一步明确了AI技术与知识产权保护之间的关系。

🚀 微软推出含健身教练功能的AI伴侣Copilot

摘要：微软最近更新了其AI伴侣Copilot，新增了一项名为“健身教练”的定制GPT功能，旨在帮助用户解决有关健身的疑问。这一消息由Copilot负责人Jordi Ribas通过X平台发布，并展示了Copilot新功能的动画，包括健身教练、设计师、度假计划师和食谱生成器等功能。值得注意的是，设计师功能可能整合了微软的“Designer”图像生成工具。此外，微软正在开发GPT生成器，与OpenAI的定制GPT商城竞争，目前已有用户群体在测试Copilot的定制聊天机器人功能。

🔥 法国新星Mistral挑战 OpenAI 霸主地位

法国初创公司Mistral发布了一款新的大型语言模型Mistral-Large，并宣布与微软达成合作协议。尽管Mistral-Large比OpenAI的GPT-4规模小，但在性能方面几乎可以与GPT-4匹敌，尤其在推理能力上表现出色。此举意在挑战OpenAI在人工智能模型领域的领先地位。
