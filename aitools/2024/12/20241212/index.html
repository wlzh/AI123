✨ 1: CodeArena

CodeArena是一个平台，让大型语言模型 (LLM) 竞赛构建相同应用，并实时显示排名。

CodeArena是一个在线平台，用于比较不同大型语言模型 (LLM) 生成代码的能力。它让多个LLM 同时构建相同的应用程序，并实时显示排名结果，方便用户观察不同LLM的性能差异。其使用场景主要在于评估和比较不同LLM的代码生成能力，为开发者选择合适的LLM提供参考，以及促进LLM技术的进步。CodeArena 使用Together AI运行LLM，Sandpack渲染UI代码，并采用Next.js、TypeScript、Shadcn UI组件和Tailwind CSS进行构建。

地址：https://github.com/Nutlope/codearena

✨ 2: ChatTTSPlus

ChatTTSPlus是ChatTTS的增强版，支持TensorRT加速、语音克隆和移动端部署。

ChatTTSPlus是ChatTTS的扩展版本，它在原有的基础上增加了TensorRT加速、语音克隆和移动端模型部署等功能。该项目使用方便，提供Windows一键安装包，并通过TensorRT实现了超过3倍的性能提升(在Windows 3060 GPU上，从28 tokens/s提升到110 tokens/s)。 它支持使用LoRA进行语音克隆，并正在开发模型压缩和加速技术以实现移动端部署。

ChatTTSPlus是一个功能强大且易于使用的语音合成工具，适用于多种场景，尤其在需要高性能和语音克隆功能的应用中具有优势。

地址：https://github.com/warmshao/ChatTTSPlus

✨ 3: Desk-Emoji

Desk-Emoji是一款开源AI桌面机器人，带表情屏、双轴控制和语音对话功能。

Desk-Emoji 是一款完全开源的 AI 桌面机器人，它拥有表情符号显示屏、双轴控制台以及大型语言模型 (LLM) 支持的语音聊天功能。用户可以通过语音与其互动。该项目提供组装指南、软件手册和固件下载，并已在淘宝开设店铺进行销售。其使用场景涵盖了AI交互式娱乐、桌面装饰以及个性化语音助手等方面。

地址：https://github.com/ideamark/desk-emoji

✨ 4: LLMOps Python Pacakge

LLMOps Python Package是一个基于Python的工具包，提供最佳实践，支持LLM模型的全生命周期管理，包含模型注册、实验追踪和部署等功能。

LLMOps Python Package是一个基于Python的工具包，旨在简化大型语言模型 (LLM) 的操作流程，它借鉴了MLOps的最佳实践，并针对LLM的应用场景进行了优化。 该工具包整合了诸多工具和技巧，涵盖了LLM生命周期的各个阶段，包括模型注册、实验跟踪、实时推理、部署、监控以及护栏设置等。

LLMOps Python Package 提供了一个全面的框架，用于构建、部署和管理LLM应用，并通过自动化和可观测性来简化LLMOps流程。 它尤其适合需要高质量、可扩展和可维护的LLM应用的团队。

地址：https://github.com/callmesora/llmops-python-package

✨ 5: python-genai

Python-genai是Google的Gen AI SDK，提供Python接口访问Google AI模型，支持文本生成、图像生成等功能。

Python-GenAI 是 Google 提供的 Python SDK，用于访问 Google 的生成式 AI 模型，包括 Gemini 等。它支持两种后端：Google AI 和 Vertex AI。 开发者可以使用它方便地调用各种大型语言模型的功能，例如文本生成、图像生成、嵌入式内容生成等。

Python-GenAI 提供了一个简洁易用的接口，使得开发者可以方便地利用 Google 的强大生成式 AI 模型，构建各种 AI 应用。 其灵活的配置和功能扩展性使其适用于各种不同的场景。

地址：https://github.com/googleapis/python-genai
