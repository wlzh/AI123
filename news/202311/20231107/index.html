🚀 OpenAI 开发者大会速览

摘要：在首届开发者大会的主要内容：

  1. GPT-4 Turbo:
  • 上下文长度：支持128K，相当于 300 页文档,
  • 输入降费 3 倍, 生成降费 2倍
  • 也更快了
  1. 更可控的输出
  • JSON 格式固定输出
  • 增强的 Function Calling（支持多个并行）
  • 种子控制（用于复现结果）
  1. 知识更新至：04/2023
  2. 开放新的 API:
  • DALLE-3 API（图像生成）
  • GPT-4V API（图像理解）
  • TTS API（语音生成）
  • Whisper V3 API（语音识别）
  1. GPT-4 支持微调，GPT-3.5 支持至 16K
  2. 自定义模型：创建具有特定领域知识的企业定制模型
  3. 提高 2 倍的速率限制，增加速率管理
  4. 版权保护（Copyright Shield）, 保护公司不受版权索赔影响
  5. GPTs: 快速自定义的 ChatGPT
  • 指令
  • 动作
  • 外部知识
  1. GPT Builder：自然语言构建自定义 ChatGPT
  2. GPT Store：即将上线，作者分成计划
  3. 助手 API：构建具有访问工具能力的自主代理（这是把 LangChain 的活给干了）
  • 线程：支持会话存储
  • 知识检索：支持分段、索引、存储嵌入
  • 代码解释器
  • 函数调用
🚀 创新工场旗下AI大模型创业公司“零一万物”完成超过10亿美元估值的新一轮融资

摘要：创新工场董事长兼CEO李开复旗下的AI大模型创业公司“零一万物”已完成新一轮融资，由阿里云领投。目前，该公司估值已超过10亿美元，成为一家独角兽企业。零一万物已推出两个开源大模型，并对学术研究完全开放，同步免费商用申请。该公司表示，AI 2.0时代将带来十倍于移动互联网的平台机会，将对软件、使用界面和应用进行重写，改变用户交互和入口。影响力：25，公众兴趣：25，新颖性：25，重要性：25。

🚀 马斯克领导的xAI发布首款产品GroK，特斯拉车或原生运行较小版

摘要：IT人工智能公司xAI发布了其首款产品Grok，由马斯克领导。马斯克暗示特斯拉的汽车可能会原生运行较小版本的Grok。用户发现马斯克点赞了一条帖子，称特斯拉汽车会运行一个缩小版、量化版的Grok模型，使用车辆本地计算能力。马斯克表示特斯拉的汽车可能拥有世界上最大的可用推理计算能力。Grok可能取代特斯拉目前的语音控制系统，xAI计划为Grok提供API，并具备音频和图像识别能力。Grok提供给X的Premium Plus订阅用户，订阅费用每月约117元。这是xAI发布的第一个软件产品。

🚀 OpenAI推出Assistants API，帮助开发者构建AI助手

摘要：OpenAI在首届开发者大会上推出了Assistants API，这是一种“专门构建的AI工具”，能够帮助开发者在自家应用程序中建立AI助手。Assistants API提供了代码解释器、检索和函数调用等功能，可以帮助开发者处理繁重的工作。该API旨在实现灵活性，适用于多种用例范围，如基于自然语言的数据分析应用、代码助理、基于人工智能的度假规划师等。Assistants API引入了持久且无限长的线程，解决了上下文窗口长度约束的问题。用户可以在Assistants Playground主页体验该API，无需编写代码。
