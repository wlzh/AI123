🦉 AI新闻

🚀 谷歌推出图像生成工具 ImageFX

摘要：谷歌在 Imagen 2 的基础上推出新的图像生成工具 ImageFX，通过简单的文字提示可以生成高质量图像。该工具包含了提示界面，让用户可以快速尝试创作和想法的相邻维度。同时，谷歌的 SynthID 技术在生成的图像中嵌入了不可察觉的数字水印，用于身份验证。这一推出激发了创造力，让用户能够探索和创新实验自己的想法。

🚀 微软员工披露OpenAI的DALL-E 3模型漏洞，能生成NSFW不当内容

摘要：微软软件工程部门经理Shane Jones发现OpenAI旗下DALL-E 3模型存在漏洞，能生成一系列NSFW不当内容。他向微软及OpenAI报告漏洞后，却被下达“封口令”，并遭到警告。Jones最终选择向外界披露相关漏洞。漏洞被披露后，有关AI生成的知名歌手Taylor Swift不雅照事件与其有关。微软回应将解决员工疑虑并修复漏洞，但声称Jones披露的漏洞成功率较低，与Taylor Swift不雅照事件是否有关尚不清楚。

🚀 微软推出Visual Studio Code 1.86新版本，增加窗口缩放和AI语音唤醒等功能

摘要：微软发布了Visual Studio Code 1.86版本，该版本包含了一系列更新和新特性，如窗口缩放改进、语音唤醒AI、多文件差异编辑器、断点依赖性测试等。该版本还改进了Markdown粘贴选项、自动保存功能，提供了更细致的扩展通知管理。此外，微软还修复了一些Bug并改进了AI的上下文处理性能。总的来说，这个新版本为用户提供了更好的编辑器体验。

🚀 亚马逊推出新AI助手Rufus，帮助改善购物体验

摘要：亚马逊推出了名为Rufus的生成式AI助手，旨在提供更好的购物体验。Rufus可以回答问题、提供建议、进行商品比较，并帮助用户完成在线购物过程。该助手已经接受了产品目录、客户评论等培训，并可以与用户进行对话，为其提供帮助。Rufus目前正在测试阶段，选定的美国客户可以在亚马逊App上进行体验。亚马逊计划在未来逐步扩大试点地区。

🚀 男子利用AI生成“造谣文章”被罚款，严重干扰社会秩序

摘要：一名男子利用AI生成“造谣文章”，通过发布在某App上获取佣金活动。他使用某款手机软件的“AI题词”功能，输入敏感字眼和时间地点，自动生成文章，其内容足够精彩以吸引读者眼球。这些文章迅速获得了近3000次浏览量，影响了社会秩序。齐齐哈尔公安机关对该男子处以罚款，并责令删除相关谣言。这是一起利用AI进行虚假信息传播的案例。对于打击网络造谣行为具有警示意义。

🔥 Qwen-VL最新升级

Qwen-VL系列多模态模型的最新升级，包括增强版本Qwen-VL-Plus和最强大版本Qwen-VL-Max。这两个版本在图像推理能力、细节识别、文本处理等方面都有显著提升，在多个任务表现出色，超越了之前的开源模型。文章还提供了实际应用示例，展示了Qwen-VL在名人识别、图像定位、视觉推理和文本处理等方面的能力。最后，文章指出了使用Qwen-VL-Plus和Qwen-VL-Max的途径，并表达了对Qwen-VL未来发展的期望。

🔥 Hugging Face推出自定义AI聊天Assistants

Hugging Face是一家位于纽约的初创公司，他们推出了第三方定制的Hugging聊天助手，用于构建个性化的AI聊天机器人。与OpenAI的GPT Builder相比，Hugging Chat Assistant是免费的，并且用户可以选择不同的开源语言模型来赋予其智能。这个举措显示了开源社区对闭源竞争对手的迅速赶超，并且展示了Hugging Face的多样化模型和框架选择的理念。虽然Hugging Chat Assistant在某些方面优于GPTs，但在其他方面还存在差距。
