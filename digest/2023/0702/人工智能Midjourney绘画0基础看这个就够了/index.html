随着 ChatGPT 的火爆全网，有关 AIGC（AI Generated Content）的相关应用也逐步走入了大众视野。其中，人工智能绘画尤为受欢迎，也越来越吸引人们的注意。在我的身边中就有越来越多的设计者和艺术爱好者想学习和应用这项技术，但却苦于目前网上的很多教程都比较零散充满了术语，有很多地方没有讲到就跳过了，不知该从何入手。

因此，我希望可以通过这套保姆级教程，让「每个人都能够轻松地掌握 AI 绘画技术」。我会用「最简单易懂的语言 + 清晰明了的图片」，让这项技术变得更加容易理解，用「理论 + 实战案例」结合的方式向前推进。好了，言归正传，让我们一起开始 Midjourney 的学习之旅吧！

首先，我们先来了解下什么是 Midjourney？

Midjourney 是一款强大的 AI 图像生成工具，具有灵活性高、易使用等特点。只需一些简短的文字描述或相关提示词它便可以将你的想象快速转化为现实，不论多么夸张或抽象。与其它 AI 图像生成器相比，Midjourney 具有更快的生成速度和更低的学习门槛，它不仅可以生成各种风格的艺术作品，还可以作为创作灵感的参考来源，因此受到了许多艺术家的青睐。

注：上面这幅曾震惊世界的作品《太空歌剧院》（Théâtre D’opéra Spatial）便出自 “Midjourney 之手”

除了《太空歌剧院》外，让我们再来看看 Midjourney 社区图库中展示的其它令人惊叹的作品（如：下图）。是的，你没看没错，它们全部是由 AI 完成的。如果你一直遵循这个教程并加以不断练习，不久的将来你也可以创作出属于自己的惊艳之作～

我相信现在你已经对 Midjourney 这款工具的概念和功能有了初步的了解。接下来，我们来看看如果想要学习使用这款工具，需要做哪些准备。

三

** 指令详解 **

经过多次版本更新和升级，Midjourney 中的各种命令操作已变得越来越多样化和便捷化，使得我们能够更加轻松地创作出惊艳的作品。但是，由于这些命令操作使用方式各不相同，如果你没有逐一了解它们，可能会错过一些很有用的功能。因此，本节内容将详细介绍 Midjourney 中所有以 “ / ” 开头的命令，帮助你更好地掌握这些命令的使用方法，让你能够更加便捷地创作出独具特色的作品。无论你是新手还是老兵，相信通过本节内容的学习，你都将有所收获。

/imagine

该命令用于图像的生成。你需要在该命令后面输入一个提示文本，Midjourney 便会基于该提示内容生成一张（四宫格）图像 。这个命令的具体用法技巧，如果看过我们上节内容的小伙伴们对它一定都不陌生了，它是进行图像创作的重要起始命令。如果你还没有学习过上一节内容，也不必担心。点击下面的链接即刻开始学习：

👉 Midjourney 人工智能艺术创作从入门到精通·基础篇2：生成你的第一幅 AI 作品

/settings

该命令用于查看和调整 Midjourney Bot 的设置。输入这个命令后会出现如下图一样，有很多按钮的面板。大家不用被吓到～哈，因为很多按钮暂时都不需要管，我们会在后面的课程中逐个讲解，目前我们只需要知道概念即可

例如：当点击选项按钮时，会出现 Current suffix:（默认的后缀），而 –v 5 则代表在你所有出图时将默认使用 V5 模型，这样你就不需要在打完提示词之后额外添加 –V5 参数后缀了

不过这几个按钮需要特殊留意，当点击 “MJ version 4 \ Base Quality \ Style Med” 两次时，可以把该项对应的默认后缀设为空。这个设计是为了给高级用户出图时增加更多的灵活性而准备的

注：这里我做了分类，同类型的按钮用相同颜色来区分

图像模型类型：

MJ version 1-5（MJ 版本1-5）

代表 Midjourney 发布的不同版本模型，V1-V3 模型已经陆续退出历史舞台，目前最常用的是 V4 / V5 模型（注：V5 模型目前只支持订阅用户使用）

Niji Mode（Niji 模型）

该模型专注于生成动漫 \ 二次元风格的图像，Niji 模型的特点颜色鲜艳、线条清晰，能够快速生成高质量的动漫图像

MJ Test（MJ 测试模型）

该模型是 Midjourney 早期的测试模型，相比 V1-V3 模型偏卡通的风格，它的成像风格介于卡通与照片之间，具有一定的艺术感和真实感

MJ Test Photo（MJ 照片风格测试模型 ）

该模型同样是 Midjourney 早期测试模型的一种，相比 MJ Test 更接近真实照片的成像风格

注：相同 Prompt 在不同模型中运行的实际效果（如上图），你可以根据自己的创作需求和喜好选择不同模型进行使用

图像质量类型：

Half Quality（半质量）

该选项会在生成图像时降低输出图像的质量，以达到快速生成的效果。速度提高 2 倍，GPU 耗时减少 ½

Base Quality（基本质量）

该选项为默认设置，它平衡了生成图像的质量与速度。在达到比较理想的生成效果的同时，对速度影响又不会太多

High Quality (2x cost)（高质量）

该选项用于生成高质量的图像，会比默认设置更加耗费计算资源。速度慢 2 倍，GPU 耗时增加 2 倍

图像风格类型：

Style Low（低风格）

该选项生成的图像较为模糊，细节不太清晰，但可以更快地生成

Style Med（中等风格）

该选项为默认设置，在生成图像速度和细节方面取得了更好的平衡

Style High（高风格）

该选项生成的图像具有更多的细节和更清晰的轮廓，但需要更长的生成时间

Style Very High（极高风格）

该选项生成的图像具有最高的细节和清晰度，但需要更长的时间生成，消耗更多的计算资源

图像展示类型：

Public mode（公共模式）

该选项为默认设置，生成的图像以及 Prompt 都会出现在画廊中，可以被其他用户查看和使用

Stealth mode（私人模式）

专业计划用户特有的展示模式，生成的全部信息都将不会被公开展示，只能在个人账户中查看和管理

特殊参数功能：

Remix mode（混音模式）

与其它参数不同，Remix mode 不会完全重置 Prompt 信息，而是在原有的基础上进行修改，这使得生成的新图像即保留了一定的原始风格和特征，同时又能够在某些方面进行改进和再创新

注：这里的操作仅仅是改变 Prompt 中的一个单词，将 “princess” 改为 “prince”。通过这一简单的替换，你将获得一个与公主风格一致的王子图像

成像效率类型

Fast mode（快速模式）

可以获得更快的生成速度，但每个月会有一定的 GPU 时间限制。一旦用完 GPU 时间，你可以在 Midjourney 网站上进行购买。使用快速模式能够极大地提高你的生产力，尤其是当你需要生成大量图像时

Relax mode（松缓模式）

在这种模式下，你生成的作品不会消耗快速 GPU 时间，但需要更长的时间才能生成图像。因此，松缓模式更适合对时间不敏感或不需要快速生成的作品

**/public 和 ** /stealth

在 “公共” 和 “私人” 模式之间切换。在公共模式下，你的作品对画廊中的每个人都是可见的（注：即使你在 Discord 中添加了自己的服务器来创建它们）。而在私人模式下，你的作品仅对自己可见，不过这个功能只有专业计划的用户才可以使用。是 /settings 中 Public mode 与 Stealth mode 按钮的快捷方式

/prefer remix

可以在不改变初始图像主体的情况下，对 Prompt 信息进行修改，从而生成一个全新的图像。使其能够在原有的基础上进行更多的探索和尝试，为你的创作提供了更多可能性。是 /settings 中 Remix mode 按钮的快捷方式

**/fast 和 ** /relax

在 “快速” 和 “松缓” 模式之间切换。在快速模式下，可以获得更快的图像生成速度，但每个月会有一定的 GPU 时间限制。一旦用完 GPU 时间，你可以在 Midjourney 网站上进行购买。在松弛模式下，你的工作不会消耗快速 GPU 时间，但生成需要更长的时间。是 /settings 中 Fast mode 与 Relax mode 按钮的快捷方式

/prefer option set

Midjourney Bot 中的一个高级命令，用于自定义生成图像时的偏好选项。这里会涉及到两个信息的定义： 1. option ：这里是为该选项命名，2. value ：这里是为该选项复值

**** 命令解释：创建一个名为 “mine” 的选项，它的参数为 –v 4 –q 2 –s 750

在实际使用 /imagine 生成图像时直接引用即可，如下图：

命令解释：/imagine prompt a cartoon panda –v 4 –q 2 –s 750

这里需要强调一点：如果将 “value” 字段设为空时，则代表删除该选项

/prefer option list

列出你使用 prefer option set 创建的所有选项， 最多可以有 20 个自定义选项

**/prefer suffix **

通过该命令可以设置一个通用后缀，使每个 Prompt 输入时都会默认添加上这个后缀，可有效提高生产力

**/prefer auto_dm **

该功能默认为开启状态，当一个任务完成时，系统会自动为你发送一条提示消息。再次键入该命令则会变更为关闭状态

/blend

它可以将 2-5 张图片进行混合，生成新的图像。是一种图生图的功能（混合技巧：为获得最佳效果，请上传与想要的结果具有相同宽高比的图像）

/show

通过唯一的作品 ID 可以将你的作品移动到其它服务器或频道，恢复丢失的作品，或刷新旧作品以生成新的变化、升级或使用更新的参数和功能（注：该命令仅适用于你自己的作品）。

获取图像 ID 的方式：登陆 Midjourney 官网，点击 “home”，找到你需要的作品 > 点击 “…” > 点击 “Copy…” > 点击 “Job ID”

/info

该命令用于查看有关你的个人资料信息

Subscription（订阅）

显示你当前订阅的计划以及下一个续订日期

Job Mode（工作模式）

显示你当前处于快速模式 \ 松缓模式。松缓模式仅对标准计划和专业计划的用户开放

Visibility Mode（可见性模式）

显示你当前处于公共模式 \ 私人模式。私人模式仅对专业计划的用户开放

Fast Time Remaining（快速时间剩余）

显示你本月剩余的快速出图的 GPU 时间。快速 GPU 时间每月重置，不可累积

Lifetime Usage（账号使用统计）

显示你 Midjourney 账号下所有图像的生成总数以及使用时长

Relaxed Usage（松缓模式使用统计）

显示你本月在松缓模式下所有图像的生成总数以及使用时长

Queued Jobs（排队作品）

显示你当前所有排队运行的作品数量

Running Jobs（正在运行的作品）

显示你当前正在运行的作品

**/subscribe **

当你在 Midjourney 网站上订阅了一个计划后，可以使用该命令生成一个个人链接，便与查看你的账户信息，包括 Midjourney 账户计划、已用时长、预计下一次续费时间等等。这个链接是个人的，请勿分享给他人

/ask

当你遇到问题时，你可以使用该命令向 Midjourney 来询问。在命令后面输入你的问题，Midjourney 便会根据你的问题寻找相关的信息并回答你。（注意：Midjourney 可能无法回答一些特定的问题，特别是需要个人专业知识的问题，建议在这种情况下还是寻求专业人士来帮助）

/help

你可以使用该命令来获取有关 Midjourney Bot 的基本操作指南和相关命令、参数、高级提示等官方文档

/invite

生成邀请链接并将其发送到你的个人消息界面，你可以发送邀请给某人，邀请其加入Midjourney
