✨ 1: Memary

Memary 是一个模拟人类记忆、增强AI代理的开源Python库。

Memary 是一个模拟人类记忆来提升人工智能代理性能的工具。其核心是通过集成多个模型（如本地运行的 Llama 和 GPT 模型）以及使用知识图谱、记忆流和实体知识存储来推进 AI 代理的记忆管理和用户偏好了解。Memary 提供了自动生成记忆、记忆模块、系统改进和回溯记忆等功能，以便在最小的开发者实现下，提升代理的智能化程度。

人工智能代理交互：Memary 可以帮助开发者构建具有记忆能力的聊天代理，使其能根据用户的历史交互内容提供更个性化和相关的响应。

知识管理与回溯：通过知识图谱管理和回溯功能，开发者可以在交互中实时注入新的知识点，也可以查看和调试代理的历史记忆。

用户偏好分析：Memary 可以跟踪和分析用户的偏好变化，帮助理解用户的兴趣点，并通过记忆模块调整代理的响应策略。

模型和工具定制：开发者可以添加自定义工具，选择适合的模型配置，并在不同配置下进行基准测试，确保系统能够在不同任务中的最佳性能。

数据注入与检索：Memary 支持不同数据格式的解析和清洗，以便更好地将私有数据注入代理记忆，同时也提供多种检索技术，以提高记忆的访问和利用效率。

地址：https://github.com/kingjulio8238/Memary

✨ 2: hero

Hero 是一个集日历、待办事项、笔记和语音助手于一身的生活管理应用。

Hero是一款在App Store上发布的全能日常助手应用，由MagicX Inc.开发，适用于iPhone。应用的评分为5.0（基于23个评价）。Hero整合了日历（与所有日历双向同步）、提醒、笔记、天气和购物订单等功能，并配备了GPT（生成式预训练模型）。其语音输入功能速度极快，可以实时更新屏幕内容，帮助用户快速记录想法。

Hero还支持用户与家人和朋友快速协调日程，分享可用时间、添加评论、共享提醒和笔记。家长们特别赞赏该应用在管理家庭事务、减少压力方面的帮助。应用需iOS 16.4或更高版本，且支持多种语言，包括简体中文。

此外，开发者承诺保护用户隐私，但部分数据可能会与你的身份关联。总体而言，Hero帮助用户更高效地管理日程和任务，是一款实用的生产力工具。

地址：https://apps.apple.com/us/app/hero-daily-assistant/id6443412722

✨ 3: OpenPerPlex

OpenPerPlex是一个使用最新技术的开源AI搜索引擎，提供强大的互联网搜索功能。

OpenPerPlex 是一个开源的AI搜索引擎，利用先进的技术提供网络搜索功能。

使用 Cohere 和 semantic-chunkers 库进行语义分块 (https://github.com/aurelio-labs/semantic-chunkers/blob/main/semantic_chunkers/chunkers/statistical.py) 使用 JINA API 对搜索结果进行重新排序 集成 Google 搜索（通过 serper.dev） 使用 Groq 作为推理引擎 支持 Llama 3 70B 模型

前端地址：https://github.com/YassKhazzan/openperplex_front 后端地址：https://github.com/YassKhazzan/openperplex_backend_os

✨ 4: lagent

轻量级的开源框架，旨在帮助用户高效创建基于大型语言模型（LLM）的智能代理

Lagent 是一个轻量级的开源框架，旨在帮助用户高效创建基于大型语言模型（LLM）的智能代理。它还提供了一些典型工具来增强 LLM 的功能。

地址：https://github.com/InternLM/lagent

✨ 5: XHand

XHand是一种实时生成高细节手部3D模型的手部虚拟形象系统。

XHand是一种实时生成高细节和照片级真实手部虚拟化身的技术，可以广泛应用于数字接口、扩展现实和游戏中，以增强用户沉浸感和自然交互体验。XHand通过三个特征嵌入模块预测手部变形位移、反射率和线性混合蒙皮权重，从多视角视频中生成高细节手部网格和渲染图像。在训练过程中，采用部分感知的拉普拉斯平滑策略，以保留必要细节并消除不必要的伪影。实验表明，XHand在InterHand2.6M和DeepHandMesh数据集上表现出色，能够实时生成高保真几何和纹理的手动画。

虚拟现实和增强现实：在虚拟和增强现实环境中，XHand可以帮助用户更自然地与虚拟物体互动，提升沉浸感。 游戏：游戏开发中可以使用XHand生成逼真的手部动画，提高游戏角色的表现力。 远程协作：XHand可以在远程协作工具中显示用户的手部动作，增强沟通的直观性。 手势识别和控制：利用XHand生成的手部模型，可以开发更精确的手势识别和控制系统，用于智能家居、车辆控制等领域。

地址：https://github.com/agnJason/XHand
