🚀 微软开源了Phi-3-mini适用于移动硬件设备

摘要：微软最新开源的小参数大语言模型Phi-3-mini，包括其架构特点、训练数据、性能测试以及未来发布计划。该模型拥有38亿参数，占用内存少，且在语言理解、编码和数学等方面表现出色。微软还计划发布更小型的Phi-3-small和Phi-3-medium模型。

开源地址：https://huggingface.co/collections/microsoft/phi-3-6626e15e9585a200d2d761e3

Ollama地址：https://ollama.com/library/phi3

技术报告：https://arxiv.org/abs/2404.14219

🚀 Llama 3 AI模型英文能力并列第一

摘要：LMSYS大模型评测社区最新发布排行榜显示，Llama 3在近90个模型中表现突出，排名第五，而在英文能力方面更是与GPT-4并列第一。排名前四的是GPT-4的不同版本和Claude 3超大杯Opus。这次评测是通过全网测评者自定题目并评分的方式进行的，Llama 3的这一成绩使Meta首席科学家LeCun与PyTorch之父Soumith Chintala纷纷表达了对Meta团队的骄傲。此外，Llama 3已经能在Groq平台上运行，表现出色。

🚀 Adobe推新版Photoshop，加入AI创作功能

摘要：Adobe近日宣布其旗舰产品Photoshop推出新版本，此次更新主要在加入了基于人工智能技术的图像编辑和创作功能，包括全新的Firefly Image 3 Foundation Model。这一模型支持高质量、多样化和细节丰富的图像生成，并能根据文本提示进行创作。新功能包括使用参考图像作为灵感、直接在Photoshop中应用文本到图像功能、生成符合现有图像风格的可替换背景以及迭代选择图像丰富创作。新版目前以Beta版本发布，预计今年晚些时候正式全面推出。

🚀 谷歌升级安卓版Gemini体验 推出浮动聊天窗口

摘要：谷歌即将对安卓版其聊天工具Gemini进行升级，改进之处在于用户现在能够在不离开当前页面的情况下，通过一个浮动窗口与Gemini进行交互和对话。这一变化解决了之前用户需跳转到一个特定的聊天窗口才能进行交流的不便之处，提升了用户体验。一个相关演示视频也被分享，展示了这项新功能的具体工作方式，通过浮动窗口的设计，用户可以更加便捷地与Gemini展开深入对话，而无需离开当前留览的网页。

🚀 Windows 11 Build 26200测试新AI功能

摘要：IT之家报道，Windows 11 Build 26200版本中出现了多项AI相关的新功能，如AI Explorer，包括Screen Capture（屏幕捕捉）、Sidebar Island（侧边栏岛）、Shoulder Tap、Screen Understanding（屏幕理解）、Intent Engine（意图引擎）和Discovery Overlay（发现叠层）。虽然这些功能的具体细节仍不明确，但它们旨在提升操作系统的智能化水平，如通过AI理解屏幕内容并提供建议。此外，新版还对硬件配置有明确要求，如ARM64处理器、至少16GB内存、至少225GB系统硬盘和配备NPU的骁龙X Elite 或X Plus。这标志着微软在加强Windows 11的AI能力，将其作为区分AI PC的重要指标。
