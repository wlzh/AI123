20240302 马斯克指控OpenAI违背成立协议，要求恢复开源


  • By AI123
  • March 2, 2024 - 2 min read



🦉 AI新闻

🚀 马斯克指控OpenAI违背成立协议，要求恢复开源

摘要：马斯克近日在旧金山高等法院对OpenAI及其CEO阿尔特曼提起诉讼，指控他们违反最初促进AI技术造福人类非营利方向的成立协议。马斯克声称，OpenAI转变为微软的闭源子公司，专注于开发AGI以最大化微软的利润，违背了初衷。他指出，GPT-4的开发背离了OpenAI的使命，成为微软专有技术，并集成到其办公软件中。目前相关方均未对此事作出回应。IT之家综述诉状要点，包括AGI的风险、和OpenAI的企业结构转变等。

🚀 音乐人包小柏借AI技术“复活”故女儿

摘要：资深音乐人包小柏运用AI技术，努力“复活”他于2021年逝世的22岁女儿。包小柏为之攻读博士、深入研究AI，希望通过这一技术让女儿在数字世界中再度“活”起来。尽管面临初期资料不足等困难，包小柏经过10个月的不懈努力，最终复刻了女儿的声纹，并能实时回应对话。他表示，AI技术成为了寄托他对女儿思念的方式，并希望通过这种方式再次听见女儿的声音，给她最完美的形象和声音。

🚀 昆仑万维发布「天工 AI」，引领AI搜索新时代

摘要：昆仑万维于2023年8月23日发布了「天工 AI」，成为国内首款AI搜索产品，以其卓越的产品体验在众多AI搜索产品中保持领先地位。该搜索引擎基于昆仑万维自研的千亿级大语言模型「天工大模型」，并融入了顶尖的MoE架构。其推理能力接近GPT-4，具有高效的训练及推理效率，强大的复杂任务应对能力以及优秀的视觉理解。此外，「天工 AI」能生成式搜索，深度挖掘用户真实意图，提供精确、相关的搜索结果，极大地提升了搜索体验。

🚀 Automattic否认向AI公司出售用户数据

摘要：近期，外媒404 Media报道称，WordPress和Tumblr的运营方Automattic计划向AI公司出售用户数据。对此，Automattic官方澄清，其默认屏蔽AI爬虫且不会分享非公开数据。报道中提到Automattic被指为OpenAI和Midjourney准备了包含Tumblr公开帖子的数据集，但此数据集被指混入了不应外泄的内容。Automattic回应表示，正在密切关注AI技术进展，探索在尊重用户隐私前提下与AI公司合作，且目前已有设置阻止AI数据抓取的选项，同时与特定AI公司合作只提供公开且未禁止分享的内容。

🚀 微软Edge测试图片可视化搜索新功能

摘要：微软正在Edge浏览器Canary版本测试一项名为“可视化搜索”的新功能，它允许用户通过右键选中图片并选择此选项，激活侧边栏Copilot进行内容处理和相关信息提供。此外，还在测试“screenshot to Copilot”功能，用户可上传截图至Copilot获取信息解答。目前，仅部分Canary用户参与测试。这表示微软正致力于通过集成人工智能技术，增强用户的浏览体验，与谷歌“圈选即搜”功能相似。

🚀 Stack Overflow 与 Google Cloud 建立合作伙伴关系

摘要：Stack Overflow 宣布与 Google Cloud 建立战略合作伙伴关系，旨在为开发者提供基于生成式AI的支持。合作内容包括在 Stack Overflow 平台上集成 Google Cloud 的 Gemini 工具，直接在 Google Cloud Console 中展示 Stack Overflow 的技术知识和编码辅助功能。这一合作有望提升开发者的效率，利用 AI 加快内容审核和优化社区参与体验。首套集成功能将于 2024 年上半年推出，展示将在 Google Cloud Next 会议上进行。此举显示了在 AI 时代，高质量和可信数据基础对技术解决方案的重要性，以及 Stack Overflow 对生成式 AI 内容管理的重视。
