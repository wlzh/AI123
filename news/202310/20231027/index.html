  • By AI123
  • October 27, 2023 - 2 min read



🦉 AI新闻

🚀 亚马逊发布卖家专用的AI工具，能为商品添加背景

摘要：亚马逊发布了一款针对卖家的生成式AI工具，允许卖家为商品添加由AI生成的背景。亚马逊称有75%的广告主因广告内容质量不佳而导致营销失败。通过比对两张广告图像，亚马逊发现带有背景的广告点击率比空白背景的高出40%。亚马逊推出的AI工具能够让卖家更轻松制作吸引人的广告图像。据悉，该工具目前处于测试阶段，但未来将扩大测试范围。亚马逊今年9月还推出了文字生成服务，卖家输入产品描述即可生成高品质的产品描述文字。根据影响力、公众兴趣、新颖性和重要性评估，该新闻获得总分80。

🚀 腾讯混元大模型升级，开放全新“文生图”功能

摘要：腾讯混元大模型近日进行了全新升级，对外开放了“文生图”功能。升级后的混元中文效果超过了GPT3.5，代码能力提升20%。腾讯通过优化算法解决了在语义理解、内容合理性和画面质感方面的技术难题，提高了模型的感知能力和生成效果。腾讯内部已有超过180个业务接入混元大模型。该模型还支持AI对话功能，用户可以申请内测体验。混元大模型的推出将在多个领域带来影响。

🚀 谷歌地图发布一系列功能更新，包括AI搜索结果和导航改进

摘要：谷歌地图将推出一系列功能更新，其中包括利用人工智能（AI）为用户带来更准确的搜索结果，根据其他用户分享的照片显示照片结果。此外，谷歌还扩展了Lens功能，利用AR和AI帮助用户找到正确的路径和方向。为车载地图做了准备，提供更详细的车道信息，辅助驾驶员规划车道变更。谷歌还将扩大对限速信息和充电站相关信息的支持。这些功能将逐渐推出到不同国家和地区，并给予各种颜色描绘建筑物帮助司机定位。

🚀 小米澎湃OS正式发布，手机端AI内测报名开启

摘要：小米澎湃OS正式发布，国内机型将于今年12月开始推送正式版，国际版机型则要等到2024年第一季度。手机端AI内测已经开启报名，并将分两期推送开发版。一期内测包括小爱输入助手、AI写真、AI搜图和实时字幕等功能；二期内测将更新小米HyperMind、端侧小爱同学文生图和端侧相册AI编辑等功能。同时，PAD端AI内测也已开放报名，支持小米Pad 6系列。此外，小米创作的AI妙画和WPS AI也均推出内测版本。

🚀 波士顿动力机器狗展现“语言能力” 成为公司“导游”

摘要：波士顿动力公司最近发布的视频展示了他们的机器狗具备了“语言能力”。这只机器狗有了能够回答问题的能力，并且还可以发出声音并开口说话。工程师们使用了 ChatGPT 的 API 和一些开源的大语言模型对机器狗进行训练，为其配备了扬声器和文字转语音功能。这只机器狗还可以充当“导游”，带领员工参观公司设施。波士顿动力公司计划通过为机器狗提供脚本和图像反馈的方式来增强它的功能。这一视频的发布引起了公众的关注和兴趣。
