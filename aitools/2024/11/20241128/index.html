✨ 1: SoraVids

Sora泄露API

SoraVids 是一个由 desuAnon 发布在 Hugging Face 平台上，旨在提供 OpenAI 视频生成模型 Sora 的生成视频及其相关提示。该项目发布了 87 个视频（约 702 MB）和 83 个对应的提示，这些内容是基于临时获得的 OpenAI API 访问权限生成的。

在发布后的几个小时内，OpenAI 撤销了该 API 密钥，导致无法再生成新视频。发布的所有视频均为 MP4 格式，帧率为 30 FPS，一些生成的提示在生成过程中可能会被修改，因此实际使用的提示可能与列出的不完全相同。

SoraVids 的内容被标记为 CC0 1.0 公共领域，旨在用于教育、艺术和研究目的。项目不隶属于 OpenAI，且原始视频都有 OpenAI 的水印以显示内容来源。

地址：https://huggingface.co/desuAnon/SoraVids

✨ 2: olmo2

OLMo 2是迄今为止最优秀的完全开放语言模型，推动了开放语言模型的发展。

OLMo 2 是当前最佳的完全开源语言模型，由 Ai2 开发。自2024年2月首次发布 OLMo 以来，开源语言模型生态系统迅速增长，开源模型与专有模型之间的性能差距逐渐缩小。OLMo 2 主要分为 7B 和 13B 两个模型，训练数据达到5万亿个标记，性能在同类完全开源模型中处于领先地位，并在某些英语学术基准测试中与开源权重模型如 Llama 3.1 竞争。

OLMo 2 的开发聚焦于提高模型训练的稳定性和效率，通过引入阶段性训练、改进的后训练方法，以及明确的评价框架，来推动模型性能的提升。此外，OLMo 2 的预训练过程经历了两个阶段，第一阶段使用广泛的多样化数据集，第二阶段结合高质量领域特定的数据集，以确保模型在各种任务上的广泛适应性。

OLMo 2 还推出了 Instruct 变体，针对指令遵循、知识回忆和推理能力进行了优化，这些改进使其在多个评估基准上表现出色，甚至超过许多开源权重模型。

此模型及其相关资源（如代码、数据和技术细节）均以开源形式提供，旨在促进研究和应用的进一步发展。

地址：https://allenai.org/blog/olmo2

✨ 3: StableAnimator

StableAnimator是一个高质量的人像动画生成框架，能保持身份一致性并无需后处理。

StableAnimator是一种创新的视频扩散框架，专注于生成高质量的保留身份人像动画。该模型通过结合参考图像和一系列姿势，能够无需后处理直接合成高保真度的视频。与现有的人像动画模型相比，StableAnimator通过精心设计的模块，确保了生成视频的身份一致性。其关键技术包括图像和面部嵌入的计算、全局内容感知的面部编码器，以及新的分布感知身份适配器，极大地提高了生成动画中的面部质量。

地址：https://github.com/Francis-Rings/StableAnimator

✨ 5: Model Context Protocol

Anthropic推出的模型上下文协议(MCP)是一种新标准，旨在连接AI助手与各种数据源，优化响应质量。

Model Context Protocol（MCP）是一个新的开放标准，旨在将人工智能助手与数据所在的系统（如内容库、商业工具和开发环境）连接起来，以帮助前沿模型生成更好、更相关的回复。随着人工智能助手的广泛应用，行业对模型能力的投资不断增加，尽管取得了快速的推理和质量进步，但最复杂的模型仍然受限于与数据的隔离。

MCP的推出旨在解决这一挑战，它提供了一个统一的标准，取代了零散的集成方式，简化了AI系统访问所需数据的过程。MCP的架构简单，开发者可以通过MCP服务器暴露数据，或构建连接这些服务器的AI应用（MCP客户端）。

地址：https://www.anthropic.com/news/model-context-protocol
