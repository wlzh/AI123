20241122 OpenAI CEO阿尔特曼年薪曝光：仅76,001美元；阿里巴巴已发布100个开源模型，积极推动AI发展


  • By AI123
  • November 22, 2024 - 2 min read



🦉 AI新闻

🚀 OpenAI CEO阿尔特曼年薪曝光：仅76,001美元

摘要：最新税务申报显示，OpenAI CEO阿尔特曼去年仅领取76,001美元薪酬，和前首席科学家Ilya Sutskever的322,201美元相差甚远。阿尔特曼创立OpenAI以来坚持低薪，称薪水只够支付健康保险。尽管年薪不高，他的财富主要来自投资，身价已达20亿美元。OpenAI在资助项目上逐渐扩大，包括伦理与新闻学奖学金。阿尔特曼的财务状况引发网友猜测，是否会通过股权间接获益。

🚀 阿里巴巴已发布100个开源模型，积极推动AI发展

摘要：在2024年世界互联网大会乌镇峰会上，阿里巴巴CEO吴泳铭宣布，公司已发布超过100个开源模型，下载量达4000万次。基于“通义千问”模型的衍生模型超过7.8万个，活跃开发者超800万，30万家企业接入AI技术重塑各行业。吴泳铭表示，市场更需针对特定领域的模型，并透露阿里云API调用价格去年下降97%，将持续降低AI算力成本，聚焦AI基础设施和开源开放。

🚀 OpenAI发布新版ChatGPT-4o重夺第一

摘要：在新加坡举办的OpenAI开发者日上，最新版本的ChatGPT-4o发布，成功夺回大模型竞技场的第一位置。新版注重提升创意写作能力，文本更具相关性和可读性，并在各项能力评分中有所提高。同时，系统新增了对敏感话题的护栏功能。虽然用户对其更新表示质疑，但部分研究显示OpenAI在持续改善其产品。用户可前往竞技场免费体验新版性能。

🚀 OpenAI计划推出自研浏览器冲击谷歌

摘要：OpenAI正考虑推出一款基于AI聊天机器人的浏览器，对谷歌发起直接竞争。公司已与康泰纳仕等多方达成协议，提供搜索服务。同时，OpenAI在ChatGPT中新增搜索功能，利用第三方技术快速提供最新信息，并与多家媒体建立合作，收集反馈以优化服务。未来几周内，这些功能将逐步开放给企业和教育用户，预计免费用户也将在几个月内获得访问权限。

🚀 微软推出全新AI智能体与Copilot功能

摘要：微软在Ignite大会上宣布，已搭建全球最大AI智能体生态，超过10万家公司使用Copilot Studio开发AI智能体，提升生产力。全新的智能体能够独立工作，支持1800个LLM模型，不再依赖OpenAI。同时，Copilot功能在Word、Excel和PowerPoint等应用中提供智能草稿、数据分析与实时翻译等功能，预计2025年全面推向市场。Copilot Pages启用多人协作，加强内容管理，推动AI在企业应用中的广泛使用。

🚀 微软发布AI Shell命令行工具

摘要：微软近日推出了AI Shell命令行工具的公开预览版，旨在为开发者和IT专业人员简化编码、排查错误和自动化工作流程。该工具结合Azure OpenAI Agent和Copilot in Azure Agent，支持自然语言查询、代码生成及提供Azure CLI和PowerShell命令建议。用户可独立安装或集成至PowerShell 7，兼容多种操作系统。AI Shell功能强大，具备即时排错、代码集成等能力，提升命令行使用体验。

🚀 马斯克旗下xAI公司融资50亿美元，估值达500亿美元

摘要：马斯克的xAI公司在最新一轮融资中筹集了50亿美元，估值达500亿美元。这轮融资使得xAI今年融资总额达到110亿美元，相较于春季的240亿美元估值，增长显著。根据CNBC报道，部分资金将用于购买10万张英伟达GPU，旨在建设Memphis数据中心，以支持特斯拉的完全自动驾驶业务。

🚀 搜狗输入法12.0版本升级，支持腾讯混元大模型

摘要：搜狗输入法于11月22日发布12.0版本，基于腾讯混元大模型，实现“输入即搜索”。新版本具备AI搜索、AI快查和AI陪伴等功能，用户可在PC端快速获取天气、股价等信息。界面设计更为清新，支持多种键盘布局和魔术手势，使用体验更流畅。此外，AI宠物功能也为用户提供丰富的互动选择。这是继7月推出多项AI功能后的再次升级。
